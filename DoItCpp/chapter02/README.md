# C++ 변수와 연산자

### \#include <iostream>
- C++에서 가장 기본적인 헤더 파일로 입출력에 대한 기능을 제공한다

### namespace
- **namespace**는 일종의 그룹으로 별칭을 이용하여 그룹 내 변수와 함수 등을 호출한다
- `std::cout`의 경우도 `std`라는 그룹에 있는 `cout`기능을 사용하고자 하는 것. 그리고 이를 이어주는 범위 지정 연산자`::`를 사용한다
- `using namespace std`를 코드 상단에 먼저 작성하면 이후 `std::`를 사용하지 않고 `cout`를 사용할 수 있다

### 지역 변수, 전역 변수
- 변수를 선언하고 사용할 때 범위가 존재한다. 
- `{ }` 을 기준으로 내부에서 선언된 변수가 있다면 해당 블록 외부에서는 사용할 수 없다. 즉 `{ }`이 지역 내에서만 사용가능한** 지역 변수**가 된다
- `int main`외부에 선언한 변수의 경우 모든 지역에서 사용할 수 있는 **전역 변수**가 된다

### L-Value와 R-Value
- L-Value는 왼쪽의 값을 의미하며 주로 대입 연산자를 기준으로 왼쪽의 값이 된다.
- int a = 1; -> 메모리 주소를 가리키는 a가 L-Value가 된다. 
- R-Value는 오른쪽의 값을 의미하며 주로 대입 연산자(=)를 기준으로 오른쪽 값이 된다.
- int b = 1; -> 단순한 값이며 메모리 주소를 담을 수 없는 1이 R-Value가 된다.
- a = b라는 식이 있다면 이때는 a, b모두 L-Value가 된다.


### 형변환 (type casting)
  #### 암시적 형변환
  - 컴파일러가 자동으로 형변환을 하는 것
  - `float` 자료형 변수가 `double`로 암시적 형변환이 일어난다
  - 더 많은 데이터를 포함할 수 있는 자료형으로 형변환을 해야 데이터가 손실되지 않고 타입이 변경된다
  #### 명시적 혐변환
  - 개발자가 의도적으로 명시하여 형변환을 하는 것
  - `(doube)15.234`이와 같이 소스코드에 명시적으로 형변환 코드를 작성하는 것을 의미한다


### keyword (키워드)
- **특별한 의미로 지정해둔 단어 이름**
- C++에는 특정 단어(이름)은 변수, 함수 등 이름으로 사용할 수 없다
- 대표적으로 if, for와 같은 단어의 경우 특정 경우에만 사용하는 단어로, 이를 변수명으로 지정할 수 없다.

### literal (리터럴)
- 리터럴은 코드에 명시되지 않은 고정된 값을 의미한다.
- 즉 int a = 1; 이라고 할 때 1이 리터럴인 것이다.
- 이 리터럴을 정확하게 명시할 수 있는데
- float f = 5; 라는 문장에서는 5는 정수 리터럴이나 컴파일 시 자동으로 float형으로 변경되어 f에 저장된다.
- 문자같은 경우에도 ''와 ""는 문자와 문자열로 구분되는 리터럴이다.
- 즉 개발자가 직업 변수에 입력한 "값"이 리터럴이라 부를 수 있다.

### 문자 리터럴 접두사
- 일반 문자 `'a'`
- 와이드 문자 `L'a'`
- UTF-8 문자 *C++17 까지* `string str2(u8"a")`
- UTF-8 문자 *C++20 부터* `u8string str2(u8"a")`
- UTF-16 문자 `u16string str2(u"a")`
- UTF-32 문자 `u32string str2(u"a")`

### 문자열 표현 방식
-  C++에서 문자열을 표현하는 방식은
-  `char *str = "Hello";`
-  `char str[] = "Hello";`
-  char를 배열 형태로 만들고, 각 요소마다 문자를 넣어 문자열을 만든다

### 삼항 연산자
- C++에서 삼항 연산자는
- `int max_number = a > b ? a : b;`
- 이와 같이 `(비교문) ? True : False` 사용할 수 있다.
- 문자 **:** 를 사용하며 |와 헷갈리지 말자
